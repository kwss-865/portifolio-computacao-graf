Tratamento de dados

O projeto "Tratamento de Dados" explora os consoles de jogos mais populares e utilizados no mundo dos games. Desenvolvido por Tufão/Kwss, ele apresenta uma análise de dados e tendências de uso, possibilitando que os usuários visualizem informações relevantes sobre os consoles. O site utiliza uma interface clara e acessível, facilitando a interação com os dados apresentados.

Floracy

O "Floracy" é um trabalho escolar que busca promover ações de caridade e apoio a causas sociais. O projeto foi desenvolvido por uma estudante como parte de um estudo sobre empreendedorismo sustentável. Ela me pediu ajuda para desenvolver uma página web para o trabalho dela.

O repositório no GitHub contém o código-fonte da página web, que inclui HTML e CSS. A página tem como objetivo incentivar a participação da comunidade em atividades sustentáveis e mostrar a importância da responsabilidade social. Você pode conferir o projeto no GitHub.

Acerte o Número

O projeto "Adivinha Número" é um jogo interativo onde o jogador deve adivinhar um número secreto entre 0 e 100. A interface é simples e amigável, proporcionando uma experiência divertida enquanto os usuários tentam descobrir o número correto.

O jogo pode incluir dicas e feedback sobre as tentativas do jogador, mas uma informação importante sobre o jogo é que foi usada uma IA de reconhecimento de voz.

Imóbiliaria Kwss

O projeto "KWSS 2.0" foi meu primeiro projeto feito em HTML, é uma landing page de uma imobiliária que ajuda os clientes a encontrar a casa dos seus sonhos. A página é fácil de usar e convida os visitantes a se inscreverem para explorar as opções de imóveis disponíveis.

Flappy Bird

O projeto "Flappy Bird" é uma recriação do popular jogo digital onde os jogadores controlam um pássaro que deve voar entre canos sem colidir com eles.

jogo é simples e desafiador, ideal para testar a habilidade dos jogadores. A interface é colorida e amigável, proporcionando uma experiência divertida.

Imóbiliaria

O código em Python é um sistema simples para gerenciar proprietários de imóveis. Ele permite cadastrar novos proprietários, listar os cadastrados e ativar ou desativar seu status. O programa usa um menu interativo e armazena os dados em uma lista de dicionários. É fácil de usar e oferece uma interface textual.
